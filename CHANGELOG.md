## V0.0.1 - MyJob 首个版本 (2026-07-14)

- 应用统一命名为 MyJob，建立 Git 版本管理规则。
- 新增公开产品首页、产品文档、更新日志和独立登录/注册页面。
- 新增日间与夜间 UI 模式，登录后工作台采用悬浮式布局。
- 新增普通用户注册登录、独立管理员后台和多维用户统计。
- 用户名调整为 8-12 位英文和数字组合，密码必须包含英文大小写、数字和特殊字符。
- 默认启用 HTTPS，本地自动生成自签名证书，生产环境支持指定可信证书。
- BOSS 直聘已接入；智联招聘、前程无忧与猎聘列入后续版本路线图。
