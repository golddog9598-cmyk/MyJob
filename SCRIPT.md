# MyJob 产品介绍口播

MyJob 是一个把岗位筛选、简历、投递、沟通和面试进度集中管理的求职工作台。

用户登录 MyJob 后，可以连接 BOSS 直聘、智联招聘、猎聘和前程无忧。工作台上方显示四个平台的汇总进度，下方按选择的平台查看岗位和沟通明细。

MyJob 的招聘平台能力全部运行在用户自己的 Chrome 或 Edge 中。浏览器扩展负责登录检测、岗位页面操作和消息同步，岗位、投递、会话与求职计划只缓存在当前浏览器 IndexedDB。服务器不会接收招聘平台 Cookie、岗位或聊天数据。

简历中心支持结构化编辑、模块排序、模板切换、多格式导入，以及 DOCX 和 PDF 导出。面向具体职位时，岗位信息先保存为浏览器本地 JD 草稿，再由用户检查和编辑。

遇到验证码、滑块、安全验证或平台风控时，MyJob 会停止自动操作并要求用户人工处理。

项目代码发布在 GitHub 的 `golddog9598-cmyk/MyJob`。
